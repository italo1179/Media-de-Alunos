Sistema de Cadastro de Alunos em C
Descrição

Este projeto foi desenvolvido em linguagem C com o objetivo de praticar conceitos fundamentais de programação, como vetores, estruturas de repetição, condicionais e cálculos matemáticos.

O programa permite cadastrar até 10 alunos, armazenando seus nomes e notas, calculando automaticamente suas médias e informando sua situação acadêmica.

Funcionalidades
Cadastro de até 10 alunos
Registro de duas notas por aluno
Cálculo automático da média
Classificação dos alunos em:
Aprovado
Recuperação
Reprovado
Cálculo da média geral da turma
Identificação da maior média da turma
Contagem de alunos aprovados, em recuperação e reprovados
Validação da quantidade máxima de alunos
Tecnologias Utilizadas
Linguagem C
Biblioteca stdio.h
Biblioteca string.h
Como Executar
Compilar
gcc main.c -o alunos
Executar
./alunos
